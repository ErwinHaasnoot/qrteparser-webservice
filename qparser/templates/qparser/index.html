Welcome to the online QRTE parser

Parsing a data file requires the following steps:

  1. Enter your e-mail address
  2. Upload your data file. The file will be parsed, this may take minutes to an hour.
  3. You will receive an e-mail when parsing is complete, this e-mail contains a unique link. Use the link within 24 hours to open a web page on which you can download your parsed data file. 24 Hours after the e-mail has been sent, the parsed data file will automatically be deleted, regardless of whether is has been downloaded.
  4. That's it!
